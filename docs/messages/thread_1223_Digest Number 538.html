From: hexacat | Date: 2005-08-18 10:32:08 UTC
This file is an Access Database that will help you keep track of
your letterbox finds and also manages your plants. You can keep a
log of your maintenance and also track your hitchhikers. You can
build mailing labels for your postal boxes. Also there is a stats
page that gives you the counts of your finds, plants, and not finds
by State. hmmm... there is also a contact module that you can build
and keep track of your letterboxing friends and a separate module
that you can track the events you attend.

this database needs Access 2000 installed on your PC in order to run
properly. Several people on the main board (letterboxing USA) use it
however, i'm not sure about the number of users in the Mid-Atlantic
group. the regular posters here are reluctant to embrace
technology?? :(

there was an earlier Excel version that was not as fancy but i have
stopped supporting that file some time ago. The Excel version did
not have nearly as many features.
