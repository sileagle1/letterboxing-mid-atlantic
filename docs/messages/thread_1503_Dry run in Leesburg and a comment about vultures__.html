Dry run in Leesburg and a comment about vultures...

From: tempusfugitmd | Date: 2006-01-09 14:54:03 UTC
Yesterday, in giddy anticipation of the Pub Crawl, Dead Duck Chuck and
I made a (relatively) dry run in Leesburg to get the lay of the land.
While we don't need much practice bending elbows, we did discover that
we may need supplemental oxygen...we come from Montgomery County, MD,
land of smoke-free restaurants.

Now, a public service announcement about vultures:

We had just enough time to search for one Leesburg letterbox. This was
in a great old cemetery, which interestingly was home to dozens of
roosting vultures. If you must search for letterboxes in the vicinity
of vultures, bring disposable shoe covers. The vile creatures puke up
pellets of undigested matter and it stinks just like you would imagine
vulture vomit to. Given the number of birds in this cemetery, there
was no avoiding the stuff. It wasn't a pleasant ride home.

Donna
