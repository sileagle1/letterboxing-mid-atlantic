There is real value in knowing how many times something has been found and how recently, as well as the number of unsuccessful attempts, but I generally favor allowing people to opt out of default rules like this.  I wouldn't want people not to post their clues over it but have no idea how strong people's views are on this subject.  I can also see why a planter might want to opt out of disclosing who has found the box on which dates (as the code currently does for Mystery boxes), but unfortunately that is not an option.
