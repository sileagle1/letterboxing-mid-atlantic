Jacobsburg EEC boxes replanted

From: Silent Doug | Date: 2004-07-26 02:29:25 UTC
Today, Jennifer and I replaced Marty's Jacobsburg boxes (which were planted
in 1999 and we've adopted). This is a two-box series where both stamps make
a single image, and the second box has been missing for quite some time
(making the series kind of pointless!). As a result, I re-carved both
stamps and rehid #2 in a new hiding place. The new clues are available here:

http://www.letterboxing.info/box/marty/jacobsburg.shtml

Note: if you ever found #1 but not #2, you'll need to find #1 again in
order to make the image -- the new stamps are not the same size as the
originals!

Doug
