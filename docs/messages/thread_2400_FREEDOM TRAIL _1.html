FREEDOM TRAIL #1

In Philadelphia, find the Founding Father Supreme who did time for debts.
13.5p WNW to Liz
7.5p NW
NNE to Founding Father Financier
ESE to South Carolina's contribution to the Constitution
NNE to Palisade

Magnetic Film Canister
It is vital that you carefully and completely replace both caps. It is a tight fit so you need to
be very careful. If this is too much for you, please skip this box.

This box is hidden in a very public place which is always beset by tourists and rangers. It is
not difficult to retrieve, but you must use some stealth and skill. If you cannot retrieve and
replace this box without being seen, please skip this box.
Stay Off The Grass
Stay Out Of The Garden
You Do Not Need To Trespass To Retrieve
This Is Hallowed Ground. Behave Yourself.
