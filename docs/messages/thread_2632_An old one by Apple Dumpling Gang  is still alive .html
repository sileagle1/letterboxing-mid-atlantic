An old one by Apple Dumpling Gang is still alive out there

From: tworstaggering | Date: 2007-08-15 21:23:51 UTC
Just FYI, a month ago I found Box #2 in a series by Apple Dumpling
Gang, "South Mountain Trinity." The logbook was nearing the pulp
stage so I took the whole thing home and dried it out the best I
could. Many of the stamps are obliterated, but most of the notes have
survived in this 2002 logbook. So if you're near Gathland State Park
in Maryland looking for something else that is terrific, one box of
this historic series is now back in place and still alive with some
new ziplocks. Great stamp! I'm sure Box #1 is long long gone, and I
certainly looked for Box #3 and couldn't find a trace [last mentioned
to be found in 2005].

And btw, you can find Box #2 for sure, but you're going to have to
think where certain things would have been five years ago, because
some things have changed...

But if "I" can find it, so can you!

Mommo
