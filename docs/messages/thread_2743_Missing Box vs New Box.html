Missing Box vs New Box

From: sewsowbizzy | Date: 2007-10-15 17:14:23 UTC
While we're on the topic of maintenance to others' boxes -- what is
the opinion of putting out a new box when the one that should be there
is obviously missing??? The new box would not be 'on topic' to the
one missing, but when posted would have the same clue (directions, not
the same story, poem, riddle). The clue would state it was a 'filler'
pending replacement of original box by original planter. Only the
original planter or their designee who is replacing the original box
could then take the 'filler' home and hopefully plant it anew or
return it.

I'm thinking this would not necessarily work or be appreciated if the
missing box was part of a series.

sewsowbizzy -- I shouldn't even be thinking of making 'filler boxes'
