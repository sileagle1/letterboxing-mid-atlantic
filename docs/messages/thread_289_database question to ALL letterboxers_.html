greetings fellow boxers.

i am curious how many boxers use the computer to track their PFX
counts. besides the paper log book you carry into the field do you
also generate a digital record as well?

i know some of you used MS Excel and others may use a database to
keep track.

i am trying to find out if there is any interest within the group (or
individuals) to beta test a database program i am writing solely for
letterboxing. the platform will be written using Microsoft Access and
Visual Basic. you will need to have MSAccess installed on your PC (if
you are using MS Office this is included in the software package).

what types of information would be useful to you? of course a PFX
count and percent successful rate. what about events? personally i've
never been to one, however, there has been a cry from other boxers
who use my Excel spreadsheet for such a feature. what about a sort
feature where you can sort by state or placer?

in regards to Exchanges, since most of those are collected at events,
should those be tracked on a separate table from the PF count? where
the event is listed and every letterboxer you exchanged stamps with
would be grouped within that event. hmmm.

Hitchhikers need to be counted, should they reside on a separate
table as well? obviously they will automatically be placed in your P
or F column regardless of which table they reside within the database.

generating a print report my prove handy but would have to be
developed later. gee, so many questions... of course, any and all
suggestions are welcome.

regards, hexacat

I just keep count in my book.
My other low-tech procedure is to print clues, write a numer in the
corner, and then put a sticky-dot with the same number on the
location of the box on a state map.
What I DO do on a spreadsheet is keep track of the condition of my
placed boxes. I enter each checkup by us, or found safe report by a
finder, with the date. When I sort by dates, I can see which box
needs attention first.
When I'm finished boxing for the day, I want to be finished, and not
have a computer task ahead of me.
