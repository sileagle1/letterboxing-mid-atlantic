Yes!  I found one of the Elm Creek Quilters books at a yard sale last summer.  Finally read it over the winter.  I really enjoyed it & want to check some others out from the library.

As for the two-color quilt block stamps - I think the series is a wonderful idea!  You could cover the color issue by doing cross-hatching in some of the pattern, dots, etc. to simulate different fabrics.  You'd get way more than two "colors" that way, especially if people use markers.

Haven't been to quilt camp myself, but it sounds like fun.  Someday, perhaps.
