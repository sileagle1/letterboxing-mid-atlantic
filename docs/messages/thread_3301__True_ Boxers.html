Since I seem to have time on my hands in front of a computer...

Can I just say that I find this name, this concept of "True Boxer" to be offensive.

I don't want to be the PC Patrol here. I just want to say that I think the diversity of the
experience of letterboxing is part of it's beauty. Some cut rubber. Some write clues.
Some like hikes. Some go for numbers. Some go to make friends. Some go to avoid
friends. Some plant more. Some reap more.

The individuality expressed in the game is, to me, as important as the game itself.

I may disagree with people, but I respect just about everybody who respects the game.
[except Cyclonic, of course. I'm sure you all understand]

So I find the idea of Uber-Boxer - as if there is some sort of standard to aspire to - to be
kind of insulting.

"opinions" for a while. Sorry if I got annoying.
