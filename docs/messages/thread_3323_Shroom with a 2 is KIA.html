It is just that, a course that disk golf is played on.

Disk golf is a game that is played like regular golf but uses frisbees,
or disks, since they are specially made for it, to play. There is a tee
and a basket that you are aiming for at the other end of the hole. It
is usually played in the woods, and there are hundreds of courses set up
nationwide. It is a lot of fun, you should give it a try sometime.
