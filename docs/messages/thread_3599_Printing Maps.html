From: sewsowbizzy | Date: 2009-07-10 20:39:10 UTC
Try this:

In Print Preview
Set for LANDSCAPE
Set Scale to say 80% - as you see the picture size shrink it brings in more of the baloons

You're not going to be able to read road names, but you'll get a general overview
Adjust Scale to how much you want to view/print
