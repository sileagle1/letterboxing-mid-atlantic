Letterboxing Database under development

From: hexacat | Date: 2004-10-28 19:23:38 UTC
Fellow letterboxers,

My letterboxing database is almost complete. Before it is released I
would like to solicit some volunteers to beta test the platform
(probably within the next month).

Some of the functionality is listed below:
PFX count (of course!) with the "PFXHE" nomenclature
PFX Log Book
Maintenance Log Book (this will include any boxes you adopted)
Hitchhiker Log Book
Event Log Book
Contact Database
Statistic Page
Export to MS Excel
Import from MS Excel (currently in development)

The PFX Log book can be sorted by ascending/descending order of any
field you choose (date, placer, Box name, Location, etc.)

The Maintenance module will keep track of all the maintenance done
(or needs to be done) on each box planted and/or adopted. Currently I
am working on a sort by "county" feature that will print out a list
of boxes planted in a specific area. I think it will help you
organize your maintenance weekends by locale.

The Hitchhiker module will keep track of all your hitchhiker travels
for each box.

The Events module simply lists all your events attended (nothing
fancy here).

The Contact module will allow you to build a list of fellow boxers
detailing their trail name (sorry, no multiple personalities
allowed!) mailing address, phone, email, and a list of their boxes.

The statistic module will give you a breakdown on your success rate
according to your attempts vs. number of finds. Also being developed
is a breakdown of the types of boxes (postal, virtual, and personal
travelers collected).

The big question is: what am I missing and what would other users
want have in a functional database????

Please contact me if you would like to be added to the distribution
list.

thanks, Hexacat
