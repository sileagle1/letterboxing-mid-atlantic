From: divinelotus73 | Date: 2005-02-27 15:17:09 UTC
So this weekend was my dad's (leafcutters) birthday, so my boyfriend
(Cactus, he's finally a letterboxer!!) planned a special day of
letterboxing. First we went to the "create an impression" letterbox
in ardmore, pa. I just wanted to say that if you get a chance to get
this box before it's gone, TRY!! It's a really cool, creative box.
Miz Scarlet it is awesome! Then we travelled out to Douglassville
for Cyclonics "Letterbox Special" box, very neat. Then we planted a
special box called The Happy Happy Birthday Box and had my dad find
it. It was so much fun, I'll be posting the clues this week so
everyone else can find it. Another great weekend of letterboxing, and
I might sneak some more letterboxing in today before the snow! Lotus
