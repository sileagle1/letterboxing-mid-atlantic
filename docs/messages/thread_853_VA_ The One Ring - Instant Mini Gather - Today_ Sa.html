We all had a fun time at the Old Brougue.  Stamped into Keep It Secret, Keep It Safe.  Met up with the Gallant Rogue, his first time meeting other letterboxers.  He only started in January yet he's got a really great sig stamp. 
 
I have now carved an event stamp using the Pub logo.  It turned out pretty good.  Well, it just looks pretty much like the logo.  But I forgot to do a date.  Does someone want to carve the date? Since the date the day before is one of those cool, once a century dates, I think we should use the date: 3/4/5 +1. That's a distinctive date that no other gather will have.  I can mail the stamp around to the participants if y'all would like.  Or maybe you'd rather I planted it. 
 
Did y'all find the Shutterbug?
